# Templates

Use these templates as your minimum operating set.

## Included templates

- `weekly-status-template.md`
- `adr-template.md`
- `risk-log-template.md`
- `portfolio-rollup-template.md`

## Recommended usage order

1. Weekly status template
2. Risk log template
3. ADR template
4. Portfolio rollup template

## Implementation tip

Start with one program for 2 to 4 weeks. Standardize quality and cadence before scaling to the full portfolio.
