# Program Data Inputs

Use this folder for program-scoped report inputs.

Recommended structure:

- `data/programs/<program-slug>/weekly-status.json`
- `data/programs/<program-slug>/adr-log.json`
- `data/programs/<program-slug>/raid-digest.json`
- `data/programs/<program-slug>/dependency-critical-path.json`

Use custom runner arguments to point report generation at these files.
