# Security Policy

## Reporting a vulnerability

If you discover a security issue related to this repository, please do not post it publicly first.

Open a private report to the maintainer with:
- Description of the issue
- Reproduction steps
- Potential impact
- Suggested mitigation (if available)

## Scope

This repository is documentation and templates focused.

Security-sensitive areas include:
- Any scripts or automation guidance added over time
- Data handling recommendations in public examples
- References to external systems that could imply unsafe defaults

## Responsible disclosure

Please allow time for triage and remediation before public disclosure.
