# Templates

Use these templates as your minimum operating set.

## Included templates

- `weekly-status-template.md`
- `adr-template.md`
- `risk-log-template.md`
- `portfolio-rollup-template.md`

## Recommended usage order

1. Weekly status template
2. Risk log template
3. ADR template
4. Portfolio rollup template

## Implementation tip

Start with one program for 2 to 4 weeks. Standardize quality and cadence before scaling to the full portfolio.

## Copilot-native usage

- Prompt-first: ask Copilot to draft from these templates and keep schema/section structure intact.
- CLI/manual fallback: edit templates directly if Copilot is unavailable.

## Email and chat report samples

If you need formatted communications, see `reports/README.md` for HTML sample reports you can adapt.
